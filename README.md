This dataset supports the analysis of the manuscript "Telecoupled Cascading Effects of a Regional Disaster on International Land Use and Biodiversity" - submitted to Nature Sustainability on 12/23/25. 

This repository contains the necessary scripts and datasets to replicate our findings and is ordered in the appropriate file structure to conduct this analysis.

Please see the associated Zenodo repository (DOI: 10.5281/zenodo.18041259) for the Source Data.
